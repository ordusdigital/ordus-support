Formulários de captura são a porta de entrada de novos leads no seu negócio. Com o Ordus System, você cria formulários personalizados que, ao serem preenchidos, adicionam o contato automaticamente ao seu CRM, disparam automações e criam oportunidades no pipeline.

Você pode incorporar os formulários em seu site, criar páginas de destino exclusivas ou compartilhá-los via link — tudo sem precisar de uma ferramenta externa.

## Passo a Passo

1. **Acesse o módulo de Sites**: No menu lateral, clique em **Sites** e selecione **Formulários**. Clique em **+ Criar Formulário**.

2. **Escolha um template**: Selecione um template pré-pronto (contato, orçamento, agendamento) ou comece com um formulário em branco clicando em **Formulário em Branco**.

3. **Adicione os campos**: Arraste e solte os campos desejados da barra lateral:
   - **Campos padrão**: Nome, E-mail, Telefone, Endereço
   - **Campos personalizados**: Crie campos únicos para seu negócio (ex.: "Quanto você fatura por mês?")

4. **Configure as opções de cada campo**:
   - Clique no campo para editar o rótulo (label)
   - Marque como **Obrigatório** para campos essenciais
   - Defina o tipo de resposta (texto, número, seleção múltipla, etc.)

5. **Configure a ação após o envio**: Em **Opções do Formulário → Ação Após Envio**, escolha:
   - Exibir uma mensagem de agradecimento personalizada
   - Redirecionar para uma URL específica (ex.: página de obrigado)

6. **Conecte ao pipeline e às automações**: Em **Opções Avançadas**, defina:
   - **Pipeline**: para qual pipeline as novas oportunidades vão
   - **Etapa**: qual etapa receberá os novos contatos
   - **Tags**: tags automáticas para segmentar o contato
   - **Campanha/Automação**: qual fluxo será ativado ao enviar

7. **Publique e obtenha o link**: Clique em **Salvar** e depois em **Publicar**. Copie o link direto ou o código de incorporação para usar no seu site.

## Perguntas Frequentes

**Como incorporo o formulário no meu site WordPress?**
Clique em **Código de Incorporação** no menu do formulário. Copie o código JavaScript e cole-o no rodapé do seu WordPress ou diretamente na página desejada usando um bloco HTML personalizado.

**Os dados do formulário aparecem automaticamente no CRM?**
Sim. Cada envio cria automaticamente um contato no Ordus System com todos os dados preenchidos. Se o contato já existir (mesmo e-mail ou telefone), os dados são atualizados.

**Posso usar CAPTCHA para evitar spam?**
Sim. Nas configurações avançadas do formulário, ative a opção **reCAPTCHA** para filtrar preenchimentos automáticos de bots.

**É possível criar formulários de múltiplas etapas (multi-step)?**
Sim. No editor de formulários, use a opção **Adicionar Passo** para criar formulários divididos em várias telas. Isso aumenta as taxas de conversão em formulários com muitos campos.
